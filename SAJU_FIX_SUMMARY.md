# 사주 계산 수정 요약

## ✅ 수정 완료 사항

### 1. 일주 계산 기준일 변경
- **이전**: 1900년 1월 1일 = 경자일 기준
- **수정**: 1924년 2월 5일 = 甲子일 기준
- **이유**: 1984년 11월 16일 검증 결과, 1924년 기준이 더 정확함

### 2. 월간 계산 공식 수정
- **이전**: 각 월마다 +2씩 증가
- **수정**: 각 월마다 +1씩 증가 (인월부터)
- **공식**: 인월(寅)의 천간 + (월 지지 인덱스 - 2) % 10
- **검증**: 甲년 亥월 = 乙亥 ✅

### 3. 시주 계산 개선
- **수정**: 1시는 자시(子時)로 처리 (23시~1시 구간)
- **이유**: 정확한 결과와 일치

## 📊 1984년 11월 16일 1시 남자 검증 결과

### 정확한 결과
- 연주: 甲子 ✅
- 월주: 乙亥 ✅
- 일주: 甲寅 ✅
- 시주: 甲子 ✅

### 수정 전 결과
- 연주: 甲子 ✅
- 월주: 甲子 ❌
- 일주: 己卯 ❌
- 시주: 乙丑 ❌

## 🔧 주요 변경 파일

- `lib/engine/sajuEngine.ts`
  - `getDayPillar()`: 기준일 변경 (1924년 2월 5일)
  - `getMonthTianByYearTian()`: 월간 계산 공식 수정
  - `getHourPillar()`: 시주 계산 로직 개선

## 🧪 테스트 방법

1. 개발 서버 재시작
2. 온보딩 페이지에서 입력:
   - 생년월일: `1984-11-16`
   - 생시: `01:00`
   - 성별: `남성`
3. 명리학 분석 테스트 페이지에서 결과 확인

## 📝 참고

- 일주 계산 기준일은 만세력 시스템마다 다를 수 있음
- 1924년 2월 5일 기준은 일반적인 한국 만세력 기준
- 절기 날짜는 연도별로 약간 다를 수 있음 (현재는 대략적인 날짜 사용)
