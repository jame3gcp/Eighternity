# 명리학 분석 시스템 테스트 시나리오

## 테스트 목표
명리학 분석 시스템의 전체 플로우를 검증합니다.

## 시나리오 1: 신규 사용자 온보딩 및 명리학 분석
1. 온보딩 페이지 접속
2. 생년월일시 입력 (1984-11-16, 01:00, 남성)
3. 온보딩 완료 후 홈 페이지 이동
4. 명리학 분석 페이지 접속
5. 명리학 분석 요청
6. 분석 결과 확인 (첫 분석 - OpenAI API 호출)
7. 동일 분석 재요청 (저장된 결과 재사용 확인)

## 시나리오 2: 저장된 분석 결과 재사용
1. 명리학 분석 페이지 접속
2. 저장된 분석 결과 조회 (GET 요청)
3. 분석 결과 표시 확인

## 시나리오 3: 오류 처리
1. 사용자 정보 없이 분석 요청
2. 잘못된 생년월일 입력
3. OpenAI API 오류 처리

## 검증 포인트
- ✅ 온보딩 플로우 정상 작동
- ✅ 명리학 분석 결과 정확성
- ✅ DB 저장 및 재사용 기능
- ✅ 오류 처리 및 사용자 피드백
- ✅ UI/UX 일관성
